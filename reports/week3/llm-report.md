# LLM Usage Report

## Tools Used

The team used ChatGPT as an AI-assisted productivity tool during Assignment 3.

## Usage

ChatGPT was used to:

* refine and improve user story wording;
* draft acceptance criteria for user stories and technical PBIs;
* review Product Backlog documentation for consistency with assignment requirements;
* assist with the preparation of Definition of Done documentation;
* help verify repository documentation structure and traceability requirements;
* support preparation of assignment-related reports.

## Human Oversight

All generated content was reviewed, edited, and validated by team members before being added to the repository.

The team remained responsible for all backlog decisions, prioritization, planning, implementation, testing, and submission artifacts.

## Limitations

AI-generated suggestions were treated as draft material and were adjusted when necessary to align with project requirements and customer expectations.
