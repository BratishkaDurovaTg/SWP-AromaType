#User Stories


##US-001

Status: Active

MoSCoW: Must Have

As a user, I want to complete a short guided questionnaire so that I can discover fragrances that match my lifestyle and preferences.

Notes: Included in MVP v1 scope.


##US-002

Status: Active

MoSCoW: Must Have

As an admin, I want to full the form, so that I can add new fragrance in data base

Notes: Included in MVP v1 scope.


##US-003

Status: Active

MoSCoW: Must Have

As a user, I want to open product card, so that I can see parfum's notes

Notes: Included in MVP v1 scope.


##US-004

Status: Active

MoSCoW: Should Have

As a user, I want to order a recommended sample set directly from the platform so that I can easily try my personalized selection.

Notes: Included in MVP v1 scope.


##US-005

Status: Active

MoSCoW: Must Have

As a user, I want to describe my style, character, and daily situations so that I can receive more relevant fragrance recommendations

Notes: Planned for future iterations.


##US-006

Status: Active

MoSCoW: Must Have

As a user,
I want to view my recommendation results
so that I can compare fragrances that match my preferences.

Notes: Planned for future iterations.


##US-007

Status: Active

MoSCoW: Should Have

As a user, I want to see why each fragrance was recommended so that I can make an informed decision before ordering samples

Notes: Planned for future iterations.


##US-008

Status: Active

MoSCoW: Could Have

As a user, I want to save my fragrance profile so that I can return to my recommendations later without repeating the questionnaire.

Notes: Planned for future iterations.


##US-009

Status: Removed

MoSCoW: Will Not Have

As a user, I want to provide feedback after testing fragrances so that future recommendations can better match my preferences.

Notes: Planned for future iterations.


##US-010

Status: Active

MoSCoW: Must Have

As an admin,
I want to edit fragrance information
so that product data remains accurate and up to date.

Notes: Planned for future iterations.
