# LLM Usage Report

## AI/LLM Tools Used

* ChatGPT (OpenAI)

## Purpose of Use

The team used ChatGPT as a supporting tool during Assignment 4. AI assistance was used for:

* improving the wording and structure of project documentation;
* reviewing and refining customer review reports, Sprint Reflection, and Sprint Retrospective;
* checking that repository documentation matched the assignment requirements;
* improving the clarity and grammar of English documentation;
* organizing README sections and report structure.

## Human Contribution

All technical implementation, project decisions, Product Backlog updates, Sprint planning, testing activities, customer communication, and repository management were completed by the team members.

All AI-generated suggestions were reviewed, edited, and adapted before being included in the repository.
