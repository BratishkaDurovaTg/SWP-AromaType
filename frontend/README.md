# Aroma Type MVP v0

The minimum technical basis of the frontend part of the Aroma Type project.

## What is displayed

- `Aroma Type`
- `Questionnaire`
- `App loaded`
- `Telegram SDK connected` or `Browser fallback mode`
- `Environment checking`
- button `Run smoke-check`
- after clicking: `Smoke-check passed`

## Health-check
