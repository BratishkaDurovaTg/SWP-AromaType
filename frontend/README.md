# Game Compass MVP v0

Минимальная техническая основа проекта.

## Что отображается

- `Game Compass`
- `Опросник`
- `App loaded`
- `Telegram SDK connected` или `Browser fallback mode`
- `Environment checking`
- кнопка `Запустить smoke-check`
- после нажатия: `Smoke-check passed`

## Health-check

После деплоя проверить `/health`.

Должно быть:

```text
OK
```
